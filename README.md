# Next.js App Template

A template for starting a Next.js web application.

## Tech Stacks

- Next.js (React.js)
- TypeScript
- Postgresql
- NextAuth
- Prisma ORM
- TailwindCSS
- TanStack Query
- Linting Rules

## Install Dependencies

```shell
npm install
```

## Run locally

```shell
npm run dev
```
